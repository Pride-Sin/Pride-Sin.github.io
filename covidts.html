About this project:

CovidTS is a web application designed for desktop with python (django framework), its used to monitor patients with covid or vaccinated, by ubication and matched with supervisors.

Main features:

  • Add patients with DNI, name, birth, current status, risk status, ubication and supervisor.
  • Add ubication with its total poblation for data analysis.
  • The architecture pattern used for this app is MVT model-view-template (somewhat "equivalent" to MVC but for django framework).
  • 100% class based views.
  • Charts showing patients and ubications status (I will try to increase the data analysis aspect in the future).
  • tables2 and django-filter django packages used.
  • Faker used for database population.

    • Stack used:

  • Django (Python)
  • MySQL
  • Bootstrap4
  • HTML5
