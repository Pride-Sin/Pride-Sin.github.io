About this project:

Documentation HERE (Github repository)


The objective of this project was making a REST API that replicates the core functions of Instagram, using the best practices for REST API design.

Features:

  • The resources from this API are USERS and PHOTOS.
  • COMMENTS and LIKES are nested resources from PHOTOS.
  • CRUD for accounts and profiles.
  • CRUD for publications (photos).
  • Comments for each photo.
  • Likes for each photo.

    • Stack used:

  • Django REST Framework (Python)
  • Postgres
  • Docker
  • GIT and Github
