About this project:

Opinar is a web application designed for mobile with python (django framework), the main feature is the creation of polls for sharing with customers/followers.

More features:

  • Creation of accounts with company/organization name, profile picture and personal information.
  • Creation of polls (questions) with their overview page, and the possibility of choosing if the results should be shown or not.
  • The architecture pattern used for this app is MVT model-view-template (somewhat "equivalent" to MVC but for django framework).
  • Mainly used CBVs with a couple of FBVs (function based views).
  • The design of the website was made only with mobile in mind, but it is usable without any problems in desktop resolution.

    • Stack used:

  • Django (Python)
  • MySQL
  • CSS with SASS preprocessor
  • HTML5
  • GIT and Github
